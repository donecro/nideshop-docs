# 简介

NideShop：基于Node.js+MySQL开发的开源微信小程序商城

* 界面高仿网易严选商城\(主要是2016年wap版\)
* 测试数据采集自网易严选商城
* 功能和数据库参考ecshop
* 服务端api基于Ｎode.js+ThinkJS+MySQL
* 基于Vue.js的后台管理系统、PC版、Wap版(开发中)
